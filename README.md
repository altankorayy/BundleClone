## Tech Used
* Designed with MVVM pattern
* Generic API requests and search
* API & Image Caching with NSCache
* Fully Unit and UI tested
* CollectionView Compositional Layout & Custom TableView
* Universal App: run on iPhone, iPad and Mac
* UIKit with no Storyboard
* Dark & Light mode support
* In-App purchases (Monthly / Annual)
* Split View and Slide Over support for iPad
* WebView and ProgressView
* SideMenu (No third-party pods)
* No external Dependencies 
* UserDefaults
* Auto layout 
* Pull to refresh
* Customized Navigation Bar
* Date Formatter
* View animations
* And moreâ€¦
